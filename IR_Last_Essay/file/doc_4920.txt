T포켓파이의 속도가 좀 궁금하실텐데요. 웹서핑하는 목적 정도로는 충분한 속도가 나오긴 합니다. SNS, 구글포토 보기 , 웹서핑 등을 자주 했는데요. 그럼 어떻게 활용하는지 살펴봅시다.     T포켓파이는 작고 동글동글하고 얇게 생겼는데요. 이걸 켜면 암호를 알고 있는 사용자는 편하게 접속해서 데이터를 사용할 수 있습니다.         T포켓파이를 그럼 열어보도록 하겠습니다. 박스 위에 붙어있는것은 유심 입니다.         제품 구성품으로는 T포켓파이 , 배터리2개, USB 케이블, 전원어댑터, 파우치, 설명서로 구성되어 있습니다.     T포켓파이 라임그린 색상 입니다. 크기는 손바닥에 딱 들어올정도의 사이즈이며, 모서리 부분이 모두 둥글게 처리되어서 동글동글하게 생겼습니다. 앞부분에는 3개의 LED가 있고 상태를 알려줍니다. LTE 신호, 전원등, WiFi 등 입니다.       후면에는 bandLTE 마크가 있을 뿐 특별한 버튼은 없습니다. 참고로 버튼은 딱 1개만 있으며 앞에 측면 부분에 있습니다. 후면 측면 부분에는 5핀 Micro USB 단자가 있어서 충전이 가능 합니다. 참고로 배터리가 부족할 경우 다른 배터리팩 등을 연결해서 충전이 가능 합니다.         커버를 분리해봤습니다. 안쪽에는 배터리와 유심을 넣는 부분이 있습니다. 참고로 내부에는 key 즉 SSID로 접근시 최초 암호가 적혀 있습니다.       유심을 장착해보겠습니다. 유심은 나노유심 입니다.       유심을 유심슬롯에 먼저 넣습니다. 상단에 보면 방향을 맞춰서 넣으라는 스티커가 있습니다. 그것에 맞게 끼우면 됩니다.       배터리를 장착 합니다. 배터리는 사진처럼 로고가 보이는 방향으로만 장착되도록 되어있습니다. 반대로 꽂고 싶어도 반대로 꽂히지 않네요.       전원을 켭니다. 빨간 화살표로 표시해 둔 부분에 전원 버튼이 있습니다. 전원 버튼을 길게 누르면 전원이 켜집니다.       간단히 속도 테스트를 해 봤습니다. 집안에서 측정시에는 다운로드 9.86 Mbps, 업로드 8.09 Mbps , 지연시간 42.8 ms가 측정이 됩니다. 참고로 최초에는 2.4GHz 기본속도로 설정이 되어있습니다.       야외에서 속도 측정을 해 보기로 합니다. 실제로는 외부에서 많이 사용하게 됩니다. 집안에서는 유무선공유기등을 사용하게 될테니 T포켓파이를 쓸일이 사실은 없죠.       암호는 라벨프린터로 출력해서 후면에 붙여놨습니다. 참고로 한번 WiFi 암호를 입력하고 접속하면 그다음에는 입력하진 않아도 됩니다. 물론 새로운 기기가 접속하려면 입력해야합니다. 참고로 WiFi 비밀번호는 설정을 통해서 변경 가능 합니다.       최초 연결을 하는 방법을 보여드리겠습니다. 처음에는 T포켓파이 전원을 켜고 모든 램프에 빛이 들어오길 기다립니다.       스마트폰에서 WiFi를 켜고 T포켓파이 신호에 연결을 합니다. 비밀번호는 배터리 커버를 열면 내부에 key 라는 부분에서 확인 가능 합니다.       WiFi 연결을 마쳤습니다. 이제부터는 넉넉한 데이터를 이용해서 편안하게 웹서핑이 가능 합니다.       스마트폰으로 웹서핑을 하거나 노트북을 이용해서 영상을 보거나 하는것은 큰 문제가 없었습니다.       길이가 긴 웹페이지도 잘 뜨네요. 웹서핑을 하고 원하는 내용을 찾는것에는 특별히 큰 문제는 없었습니다.         그런데 속도가 궁금했습니다. 먼저 기본상태에서 속도 측정을 해 봤습니다. 다운로드 12.2 Mbps, 업로드 13.8 Mbps가 측정 됩니다. 물론 속도는 그렇게 빠른 편은 아닙니다. 실제로 WiFi 가 아니라 LTE로 스마트폰에서 직접 벤치비를 돌려보니 78.1 Mbps (다운로드속도) 가 측정이 되더군요. 물론 이것은 2.4GHz 기본값으로 설정된 상태에서 측정한 결과 입니다. 그런데 5GHz 고속으로 변경을 한 상태에서도 속도는 크게 다르진 않았스빈다. (물론 지역에 따라서는 다를 수 있습니다.) 그런데 속도가 빠르면 데이터를 소모하는 속도도 무척 빨라집니다.       구글포토는 제가 참 많이 이용하는 서비스 입니다. 카메라로 찍은 이미지를 모두 업로드 하도록 해놓았고, 스마트폰이나 태블릿을 찍은 이미지도 모두 다 업로드 되도록 해놓았습니다. 사진이 많이 쌓여갈 수 록 추억도 쌓여가는데요. 필요한 이미지를 찾기 위해서 구글 포토에서 이미지를 넘겨서 봤습니다.       집근처 맛집에가서 사진을 찍어뒀던 사진인데요. 사진을 열어서 보거나 웹서핑 하는 정도는 큰 문제가 없네요.         설정페이지로 접근하기 위해서는 T포켓파이에 연결이 된 상태에서 브라우저에서 http://192.168.1.1 을 입력하여 접근 합니다. 이렇게 하면 게이트웨이 즉 T포켓파이에 연결이 되어서 설정페이지가 나타납니다. 최초 연결을 위해서는 아이디는 user 이며 비밀번호는 0000 입니다.   요약정보, WiFi설정, 고급설정, 기기설정으로 크게 나뉘어져 있습니다.           WiFi 설정 - 공통 설정에서는 속도 설정과 신호의 세기 등을 설정할 수 있습니다. 그런데 실제로 해보니 지역에 따라 다르긴 하겠지만 5GHz 고속으로 설정해도 2.4GHz 기본과 속도 차이가 크진 않았습니다. 그런데 생각해보면 T포켓파이를 사용하는것도 데이터를 좀 더 사용하기 위해서인데요. 여러명이 동시에 연결해서 사용할 수 있는 제품이다보니 속도가 너무 빠르다보면 데이터 소모량이 너무 빠릅니다. 그래서 오히려 여러 기기들을 모두 연결할 수 있게 2.4GHz 기본으로 놓는게 더 좋아보입니다.       기기설정에서는 비밀번호 변경, 초기화 설정 등을 할 수 있습니다. 유심 잠금 등도 설정이 가능 했습니다.           이 외에 가장 중요한 부분은 데이터 사용량 확인하는 부분 입니다. 데이터 사용량 부분을 눌러보면 , T포켓파이를 통해서 얼마만큼의 데이터를 사용했는지 확인할 수 있습니다. 그리고 지정된 데이터 사용량보다 더 많이 사용하는것을 막기 위해서 사용제한 사용도 설정할 수 있었습니다.   T포켓파이는 시골집에 갔을 때 유용하게 사용할 수 있을거란 생각이 드네요. 시골집에는 인터넷이 안되기 때문이죠. 여러명이 스마트폰으로 연결해서 사용할 수 있어서 가족 모두 연결해서 사용할 수 있을 거란 생각이 드네요. 그리고 인터넷 라인이 들어가기 힘든 장소에서 사용하면 유용하겠다는 생각이 들었습니다. 시골집에서 속도 측정하게 되면 그 결과도 올려보도록 하겠습니다.   참고로 8월 30일까지 KTX 서울역 & 부산역 (12:00~18:00)에서 매일 T포켓파이 런칭 행사를 진행될 예정입니다. 다양한 행사와 경품이 준비되어 있으니, 한번씩 방문해주시는 것도 좋을 것 같습니다.         인증샷 한장으로 T포켓파이 1대 주는 이벤트도 진행 중이네요. http://www.tpocketfi.com/?mn=3  응모 한번 해보세요.     카카오스토리 트위터 페이스북 '얼리어답터_리뷰 > 스마트기기' 카테고리의 다른 글 갤럭시노트5 리뷰 카메라 기능 신기능 비디오포털  (5) 2015.09.25 워터클 물 많이 마시는 습관 건강해지는 텀블러  (2) 2015.09.19 T포켓파이 후기 속도 및 사용량 확인 방법  (0) 2015.09.07 유플러스티비링크 U+tvLINK 유플릭스 영화 보기 미라캐스트  (3) 2015.08.05 레이저 나부X 스마트밴드 NaBu X 후기  (0) 2015.07.16 스카이라이프 UHD 설치 고화질 4K 컨텐츠 후기  (8) 2015.06.30 Tweet TAG It, T포켓파이, T포켓파이 사용량 확인, T포켓파이 속도, T포켓파이 후기, 방법 이 글은 "씨디맨" 의 동의 없이 전문 재배포 금지. 링크 및 트랙백은 허가 없이 무제한 허용 *  [자세히보기] 티스토리 댓글 : 0 개 SNS댓글 : 개 트랙백 : 0 개 질문 또는 댓글을 남겨 주세요. 질문글은 무조건 답변 드리겠습니다. 이곳에 질문을 올려주세요. 이 글과 연관이 없는 급한 질문은 빠른 질문/답변을 이용해 주세요! 댓글 입력 폼 Name * Password Website Secret                                   [무료이미지 업로드 방법 설명] Please enable JavaScript to view the comments powered by Disqus. 트랙백 Trackback Address :: http://cdmanii.com/trackback/4920 Prev 1 ... 107108109110111112113114115 ... 3086 Next 씨디맨 연락처, 리뷰 의뢰 문의 cdmaniipch@gmail.com 010-3066-8471 (정말 급한 질문은 전화로 해주셔도 됩니다. 다만 가능하면 질문/답변란을 이용해주세요.) recent post 2015 우수SW전시회 웹캣2.0 시큐메일가드...
