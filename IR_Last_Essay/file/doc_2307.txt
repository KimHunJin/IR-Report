XE2 스마트하드 성능편 Posted by 씨디맨 Posted in " 얼리어답터_리뷰 "   2011.06.25 17:08 스마트하드 3.0!! 성능에 대해서 알아보도록 합시다. USB3.0 하드디스크의 성능을 보여드리는 시간이기도 합니다. 대부분의 USB 3.0 인터페이스를 가진 외장형 하드디스크가 비슷한 성능을 보여준다는것도 참고하시기 바랍니다. 그 이외에 부가기능등에 대해서도 알아보도록 하겠습니다. 스마트하드 3.0!! 는 이름에서도 알 수 있듯 가장 큰 특징은 USB 3.0 입니다. 저도 USB 2.0 하드디스크를 여러개 가지고 있지만 한가지 단점이라면 실제 전송 속도가 느리다는 문제가 있죠. 용량 큰 파일을 백업하거나 가져갈 때 시간이 많이 걸린다는 단점이 있습니다. 물론 시간이 좀 더 걸리는것 외에는 큰 차이점이 없긴 하지만 이 시간이라는것이 상당히 중요할 때가 있죠. 그런 문제도 E-SATA 같은 인터페이스도 있긴 하지만 그런 인터페이스를 사용하기 위해서는 단가가 올라가는 문제도 있고 컴퓨터가 해당 인터페이스를 지원해야만 하는 문제도 있죠. USB 는 이름에서도 알 수 있듯 범용입니다. 대부분의 컴퓨터에 단자가 있습니다. 때문에 어디서든 사용할 수 있다는 장점이 있죠. USB 3.0 인터페이스 외장형 하드디스크를 쓰기 위해서 반드시 USB 3.0 인터페이스에 연결할 필요는 없습니다. 다만 최대의 속도를 보장 받기 위해서는 USB 3.0 에 꽂아야하죠. USB 는 하위 호환을 지원하기에 USB 3.0 이라는 뜻은 USB 3.0 (2.0 이하 호환) 이라는 뜻이기도 해서 USB 2.0 메인보드에도 연결해서 사용이 가능 합니다. 물론 전송 속도는 낮아지겠지만 사용이 가능하다는 뜻이죠. 이런 이야기를 길게 하는 이유가 실제로 이런 문제로 일부러 USB 2.0 외장형 하드디스크를 구매하는분도 있었기 때문입니다. 가격이 비슷하다면 USB 3.0 인터페이스를 가진 장치가 여러모로 유리하다는 점도 알아두도록 합시다. 스마트하드 3.0!! 와 USB 2.0 비교 스마트하드 3.0!! 입니다. 케이블도 연결을 해둔 상태 입니다. 성능을 알아보기 위해서 USB 2.0 인터페이스를 가진 외장형 하드디스크와 성능 벤치 비교를 해보도록 하겠습니다. 왼쪽은 스마트하드 3.0!! 이며 오른쪽은 USB 2.0 인터페이스를 가지고 있는 외장형 하드디스크 입니다. 위는 비교대상의 USB 2.0 인터페이스 외장형 하드디스크이고 아래는 스마트하드 3.0!!  입니다. 단자부분이 다른점 빼고는 서로 기능상 차이점은 크게 없습니다. 오른쪽에는 상태표시등이 있습니다. 두께는 두꺼운건 아니지만 약간 있는 형태 입니다. 스마트하드 3.0!! 와 USB 2.0 외장형 하드디스크의 성능을 알아보기 위해서 둘다 USB 3.0 인터페이스에 연결을 했습니다. 공정한 테스트를 위해서 같은 컨트롤러내에서 체크를 하기 위해서이죠. 물론 USB 2.0 외장형 하드디스크를 USB 3.0 인터페이스에 연결했다고 해서 USB 3.0 으로 동작하는건 아닙니다. USB 2.0 으로 동작을 하죠. 스마트하드 3.0!!  와 비교대상의 하드디스크를 동시에 구동을 했습니다. 소음은 둘다 작네요. 스마트하드의 조건에 조용한 하드디스크라는 내용도 있었는데 조용합니다. 특별히 제가 하드디스크의 소음이 거슬려서 빼놓지 않은것만 봐도 알 수 있었죠. USB 2.0 외장형 하드디스크 경우 꾸준하게 35MB/sec 정도의 성능을 보여줍니다. 스마트하드 3.0!! 경우에는 최대 89.4MB/sec 에서 평균 71.0MB/sec 정도의 높은 성능을 보여줍니다. 물론 스마트하드를 분해해서 하드디스크를 S-ATA 인터페이스에 연결하면 더 높은 성능을 보여줄겁니다. 다만 USB 3.0 이라는 범용장치를 사용한다는 점에 잇점이 있는것이죠. 단순히 비교하면 35.6MB/sec VS 71.0MB/sec 이기때문에 2배의 읽기 성능 차이가 있습니다. 쓰기 속도도 비슷하기에 파일을 옮기거나 쓸 때 2배의 차이가 있다는걸 알 수 있죠. 작은 파일은 옮길때 큰 차이를 느끼지 못할테지만 큰용량의 파일을 옮길 때 40분이 걸리는것과 20분이 걸리는것은 생각보다 큰 차이이기 때문에 스마트하드 3.0!! 에는 장점이 있습니다. 스마트하드 3.0!! 에 파일을 실제로 전송을 해봤습니다. 79.1MB/sec 정도가 나오네요. 물론 파일의 단편화에 따라서 성능차이는 생길 수 있지만 기존의 USB 2.0 하드디스크와 차이는 분명 있네요. USB 2.0 인터페이스를 가지고 있는 외장형 하드디스크 실제 전송 화면 입니다. 27.7MB/sec 정도가 나오네요. 실제로 사용을 해보면 USB 3.0 인터페이스를 가진 장치가 효용성이 높다는걸 알 수 있습니다. 그리고 앞으로도 점점 활용성이 더 높아진다는 장점도 있죠. 스마트하드 3.0!! 최초 장착 스마트하드 3.0!! 를 컴퓨터에 연결 후 용량을 체크한 부분입니다. 용량이 1TB 보다 약간 작게 표기되는 이유는 1MB 의 단위를 1000KB 와 1024KB 로 표기에서의 단위와 실제 사용용량의 단위가 다르기 때문입니다. 스마트하드 3.0!! 부가기능 스마트하드 3.0!!  에는 부가기능이 있습니다. 그래서 최초에 스마트하드 3.0!! 를 장착해보면 포멧할 필요가 없이 미리 NTFS 파일시스템으로 포멧이 되어있습니다. 프로그램이 내장되어 있기 때문이죠. 내장되어 있는 파일은 백업프로그램인데 자동화 백업이 가능 합니다. 중요한 파일을 2중 3중으로 백업을 하는분이 분명 있습니다. 그런 분들을 위한 편리한 프로그램이죠. 한글로 되어있고 사용방법은 어렵진 않았습니다. 물론 한글로 되니 메뉴얼이 내장되어 있습니다. 편리한 사용을 돕는 내용이므로 필요하신분이 사용하면 됩니다. 스마트하드 3.0!! 총평 스마트하드 3.0!!를 사용해보면서 느낀점은 USB 3.0 의 사용이 앞으로 점점 많아질것이고 편리해질것이라는 생각이었습니다. 막상 지금 사용하는 컴퓨터를 포멧하기 위해서 데이터를 백업하려면 시간이 많이 걸렸었는데 이제는 파일 복사 속도가 빨라져서 시간을 아낄 수 있게 되었습니다. 물론 용량도 커져서 좀 더 많은 파일을 백업할 수 있게 되었습니다. 물론 시간이 지나면 좀 더 용량이 큰 스마트하드가 나올것입니다. 물론 USB 3.0 의 하드디스크로 나오게 될것이죠.
